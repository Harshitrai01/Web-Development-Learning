Units in CSS

  1. Absolute Unit - Size of image is fixed on any device (either phone or laptop).
    • mm
    • cm
    • in
    • px
  2. Percentage Unit
    • div{ width:10%; } - This width is applied on parents pixel.
      • Suppose div is inside a parent having width = 500px. So width of div will be 10% of 500=50px.
  3. Relative Unit
    1. Relative to Font Size
      • em - Relative to parent element font size.
        • If parent font size is 8px. Then 2em will be equal to 2*8=16px.
      • rem - Relative to root element that is html.
    2. Relative to Document/View Port All area which is visible is viewport.
      • vw - 1/100 th of view port width.
      • vh - 1/100 th of view port height
